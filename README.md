# Customer Churn Analysis

## Objective

Analyze customer churn patterns and identify factors affecting customer retention.

## Dataset

IBM Telco Customer Churn Dataset

## Tools Used

* Python
* Pandas
* VS Code

## Key Findings

* Overall churn rate: 26.5%
* Month-to-month contracts show highest churn.
* Fiber optic customers churn more than DSL customers.
* Customers without tech support are more likely to leave.
* Electronic check users have the highest churn rate.
* Gender has minimal impact on churn.

## Recommendations

* Promote long-term contracts.
* Improve support services.
* Review fiber optic customer experience.
* Encourage automatic payment methods.

## Conclusion

Contract type, technical support, internet service, and payment method are major factors influencing customer churn.
